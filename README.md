"길 가는 사람 붙잡고 물어봐라." 사람들은 흔히 자신의 의견이 보편적이란 걸 피력할 때 이런 말을 한다. 아무 사람이나 붙잡고 물어봐도 나와 같은 대답이 나올 거라는 자신감이 바탕될 때 종종 소환되는 '길 가는 사람들', 과연 우리는 같은 생각을 하며 살까? 나는 다수에 속할까? 이러한 의문과 궁금증들을 해소해주기 위해 <길 가는 사람들>을 개발한다.
